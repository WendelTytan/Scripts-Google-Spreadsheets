Scripts para o Google SpreadSheets

listarArquivos = ele pesquisa aquivos dentro de uma pasta compartilhada do google drive, 
listando nas colunas A/B/C os (em ordem respectiva) Nome/Link no Drive/Nome da pasta onde está, 
os arquivos vão ser listados por ordem de verificação.
onde:   B1 -> você digita o ID da pasta que deseja escanear.
no codigo tem que informar qual a aba em que ele vai imprimir as informações
as informações segundo o codigo, serão impressas apartir de A/B/C:4, pois A/B/C:3 ficam os cabeçalhos


pesquisarArquivos = ele pesquisa aquivos dentro de uma pasta compartilhada do google drive, 
listando nas colunas A/B/C os (em ordem respectiva) Nome/Link no Drive/Nome da pasta onde está, 
os arquivos vão ser listados por ordem de verificação, e ele também tem uma função para pesquisar por arquivos
por arquivos com nome especifico.
onde:   B1 -> você digita o ID da pasta que deseja escanear.
        B2 -> você digita o nome que quer pesquisar
no codigo tem que informar qual a aba em que ele vai imprimir as informações
as informações segundo o codigo, serão impressas apartir de A/B/C:4, pois A/B/C:3 ficam os cabeçalhos
